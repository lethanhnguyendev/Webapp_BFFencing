Gallery images are organized in subfolders. Each subfolder = one category.

Structure:
  gallery/
    colorbond Good Neighbour/  - Good Neighbour fence projects
    fence extension/          - Fence extensions
    gates/                   - Gate installations
    post and rail colorbond/  - Post and rail fences
    retaining wall/           - Retaining wall projects
    slat/                    - Slat fence projects
    tubular/                 - Tubular blade fences

Add new categories: create a subfolder + add it to src/gallery-config.js
